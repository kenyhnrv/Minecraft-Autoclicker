### Install 🔧
```

```

## Usage 💻
```
Enter the CPS

F4 -> Self Destruct
F5 -> Show and Hide the Window
F6 -> Switch to Alternative Mode (Break Blocks and Eat)
F7 -> Switch to Normal mode
```
